# Shiva Travels Website

Live Demo: [https://shiva-flax-ten.vercel.app/](https://shiva-flax-ten.vercel.app/)

A premium, responsive, and feature-rich Next.js website for **Shiva Travels**, a professional car rental and travel agency. 

## Features
- **Modern UI/UX**: Premium styling, glassmorphism elements, dark/light theme options, and smooth animations using Framer Motion.
- **Fare Estimator**: Interactive calculator to estimate tour/rental charges based on trip type, vehicle choice, and distance.
- **AI Recommendation Engine**: Smart suggestions for travelers choosing vehicle classes and destination routes.
- **Interactive Booking Form**: Form with validation, booking status tracker, and downloadable receipt generation.
- **Multilingual Support**: English, Telugu, and Hindi translations.
- **Database fallback**: Integrates with a Supabase PostgreSQL database for persistent storage, falling back to LocalStorage in offline or serverless static-only environments.

## Technologies Used
- **Framework**: Next.js (App Router)
- **Styling**: TailwindCSS
- **Animations**: Framer Motion
- **Database**: PostgreSQL (via Supabase)

## Getting Started

First, install dependencies:
```bash
npm install
```

Second, run the development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
